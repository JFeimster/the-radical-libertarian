# Production Assets

This folder stages final production image slots for The Radical Libertarian.

## Rule

These slots do not auto-inject into the homepage or page layouts. Replace files only after confirming dimensions and visual fit.

## Folders

- `hero/` — 1600x900 hero/page art
- `social/` — 1200x630 Open Graph cards
- `thumbnails/` — 1280x720 YouTube/podcast thumbnails
- `square/` — 1080x1080 profile/social posts
- `covers/` — wide cover images

## Important

Do not add oversized images directly into `index.html` without layout testing.
