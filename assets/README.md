# Assets

Visual assets for The Radical Libertarian.

## Folders

- `brand/` — logos, marks, wordmarks
- `icons/` — app icons and section icons
- `images/` — hero and page image placeholders
- `social/` — Open Graph and social-share images

## Current Direction

Brutalist punk-libertarian editorial system:

- cream paper background
- black borders
- red accents
- hard offset shadows
- bold uppercase type
- zine texture
- no official party branding
- no fake seals
- no fake endorsements

## Recommended Production Formats

The repo currently uses SVG placeholders for easy static hosting and version control.

When final images are created, export:

- Open Graph: 1200x630 PNG/WebP
- YouTube thumbnail: 1280x720 PNG/WebP
- Square social: 1080x1080 PNG/WebP
- Notion/wide cover: 1600x400 PNG/WebP
- Hero art: 1600x900 PNG/WebP
