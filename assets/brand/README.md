# Brand Assets

## Files

- `trl-logo-box.svg` — boxed T.R.L. logo
- `trl-mark.svg` — circular/arrow mark
- `trl-wordmark.svg` — full wordmark

## Usage

Use the boxed T.R.L. logo in headers and compact brand moments.

Use the wordmark for wide banners, media kit pages, podcast artwork, and social headers.

Use the mark when a compact avatar or app-like symbol is needed.

## Restrictions

Do not use official Libertarian Party marks unless explicitly allowed.

Do not create fake official seals, fake badges, fake endorsements, or fake institutional credentials.
