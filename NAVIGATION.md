# Navigation

## Primary Header Nav

Keep primary navigation lean:

- Home
- Essays
- Manifesto
- Podcast
- Join

The header is for conversion and orientation, not every page in the republic.

## Secondary Navigation

Use secondary/footer/internal links for:

- Start Here
- Explore
- Search
- Sitemap
- Library
- Tools
- Resources
- Archive
- Assets

## Utility Pages

- `search.html` — lightweight static search
- `sitemap.html` — human sitemap
- `explore.html` — guided browsing page
- `navigation.html` — navigation system overview
- `updates.html` — build/update log

## Data Files

- `data/navigation.json`

- `data/page-groups.json`

## Rules

- Keep top nav small.
- Add deep links through footer and context sections.
- Link essays to relevant tools and podcast pages.
- Link tools back to essays/resources/join.
- Keep the sitemap human-readable.
