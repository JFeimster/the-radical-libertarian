# Transcript Workflow

## Steps

1. Get source URL or audio/video file.
2. Generate raw transcript.
3. Clean transcript.
4. Add title, date, source, and summary.
5. Extract 5–10 clips.
6. Publish transcript page.
7. Link to podcast episode, essay, and newsletter.

## Output Fields

- Title
- Source URL
- Transcript
- Summary
- Pull quotes
- Clip ideas
- Related links
