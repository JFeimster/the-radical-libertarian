# Design System

## Brand Direction

The Radical Libertarian is brutalist, editorial, loud, stripped-down, punk-philosophical, and intentionally anti-institutional.

It should feel like:

- A manifesto stapled to a courthouse door
- A zine with better typography
- A podcast flyer from the parallel economy
- A political field manual that learned CSS

## Visual Principles

### 1. High Contrast

Use:

- Cream background
- Black text and borders
- Red emphasis
- Yellow interaction states
- Hard shadows

### 2. Hard Edges

No soft SaaS goo.

Use:

- Thick borders
- Offset shadows
- Sticker labels
- Brutalist cards
- Oversized headlines

### 3. Editorial Scale

Headlines should dominate.

Use:

- Impact / Arial Black style display type
- Courier / monospace for labels and badges
- Serif body copy for a pamphlet / essay feel

### 4. Texture

Use subtle grain, diagonal marks, and print-like imperfections.

## Component Rules

### Header

- Logo left
- Main nav right
- Sticky if usable
- Thick bottom border

### Cards

- Cream or red background
- Thick black border
- Hard offset shadow
- Big uppercase heading

### Buttons

- Red by default
- Thick black border
- Hard shadow
- Move on hover

## Avoid

- Corporate gradients
- Generic startup blue
- Soft rounded SaaS cards
- Stock political imagery
- Flags everywhere
- Fake official party styling
